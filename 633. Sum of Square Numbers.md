### 题目
[633. Sum of Square Numbers]()
### 思路
352ms 5.43%

用哈希表存储从0-sqrt的所有平方数，然后再次遍历。
### 代码
```c++
class Solution {
public:
    bool judgeSquareSum(int c) {
        unordered_set<int> us;
        for(int i=0;i<=sqrt(c);i++){
            us.insert(i*i);
        }
        
        for(int i=0;i<=sqrt(c);i++){
            if(us.find(c-i*i)!=us.end())
                return true;
        }
        return false;
    }
};
```
