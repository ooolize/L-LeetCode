### 题目
[397. Integer Replacement](https://leetcode-cn.com/problems/integer-replacement/submissions/)
### 思路
error: 超时

刚开始打算就动态规划， O(n)就好了，结果20/47，测试量还是大啊。

### 代码
超时
```c++
class Solution {
public:
    int integerReplacement(int n) {
         vector<int> v(n+1,0);
        v[1]=0;
        
        for(int i=2;i<=n;i++){
            if(i%2==0) v[i]=v[i/2]+1;
            else
                v[i]=min(v[i-1],v[(i+1)/2]+1)+1;
        }
        return v[n];
    }
};
```
