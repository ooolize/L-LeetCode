### 题目
[263. Ugly Number](https://leetcode-cn.com/problems/ugly-number/submissions/)
### 思路
8ms 38.03%

从大到小，挨个剔除5,3,2因数。最后如果剩余1，那就对了。


### 代码
```c++
class Solution {
public:
    bool isUgly(int num) {
      if(num<=0) return false;
      while(num%5==0) num/=5;
      while(num%3==0) num/=3;
      while(num%2==0) num/=2;
      return num==1;
    }
};
```
