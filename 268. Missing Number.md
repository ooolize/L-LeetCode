### 题目
[268. Missing Number](https://leetcode-cn.com/problems/missing-number/submissions/)
### 思路
44ms 26.77%

求nums和，然后减[0,n+1]的和，取反。

32ms 51.15%

一次遍历,加的同时减
### 代码
```c++
class Solution {
public:
    int missingNumber(vector<int>& nums) {
        int count=0;
      for(auto p:nums)
        count+=p;
      for(int i=0;i<=nums.size();i++)
        count-=i;
      
      return abs(count);
    }
};
```

class Solution {
public:
    int missingNumber(vector<int>& nums) {
        int count=0;
      
      for(int i=0;i<nums.size();i++){
            count+=i-nums[i];
      }
      count+=nums.size();
      return count;
    }
};
