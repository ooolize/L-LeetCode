### 题目
[Counting Bits.](https://leetcode-cn.com/problems/counting-bits/)
### 思路
76ms 98.16%

遍历数一,用了bitset


### 代码
```c++
class Solution {
public:
    vector<int> countBits(int num) {
        vector<int> res;
        bitset<32> b;
        for(int i=0;i<=num;i++){
            b=i;
            res.push_back(b.count());
        }
        return res;
    }
};
```
