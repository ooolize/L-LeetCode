### 题目
[961. N-Repeated Element in Size 2N Array](https://leetcode-cn.com/problems/n-repeated-element-in-size-2n-array/submissions/)
### 思路

### 代码
```c++
class Solution {
public:
    int repeatedNTimes(vector<int>& A) {
        unordered_set<int> us;
        for(auto p:A){
            if(us.find(p)!=us.end())return p;
            else us.insert(p);
        }
        return -1;
    }
};
```
