### 题目
[279. Perfect Squares](https://leetcode-cn.com/problems/perfect-squares/submissions/)
### 思路
error: 超时 513/588

动态规划，a[i]=min(a[i],a[j]+a[i-j])，新的数由旧的数相加而成。

312ms 10.75%

改进的动态规划，在i之前寻找j时不再从[0,i-1]而是[1,sqrt[i]],这样我们就能达到O(n^3/2)
### 代码
```c++
class Solution {
public:
    int numSquares(int n) {
        vector<int> a(n+1,INT_MAX);
        a[0]=1;
        
        for(int i=1;i<=sqrt(n);i++){
            a[i*i]=1;
        }
        
        for(int i=2;i<=n;i++){
            for(int j=i-1;j>0;j--){
                a[i]=min(a[i],a[j]+a[i-j]);
            }
        }
        return a[n];
    }
};
```
