### 题目
[477. Total Hamming Distance](https://leetcode-cn.com/problems/total-hamming-distance/)
### 思路
暴力法O(n^2)超时，41/47


### 代码
暴力法
```c++
class Solution {
public:
    int totalHammingDistance(vector<int>& nums) {
        int count=0;
        for(int i=0;i<nums.size();i++){
            for(int j=i+1;j<nums.size();j++){
                bitset<32> b(nums[i]^nums[j]);
                count+=b.count();
            }
        }
        return count;
    }
};
```
