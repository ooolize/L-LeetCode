### 题目
[202. Happy Number](https://leetcode-cn.com/problems/happy-number/)
### 思路
4ms 87.97%

保存每次拆分求和的结果，如果出现重复说明不成立，如果为1说明是快乐数。
### 代码
```c++
class Solution {
public:
    int total_sum(int n){
        int tmp=0;
        while(n>0){
            tmp+=(n%10)*(n%10);
            n/=10;
        }
        return tmp;
    }
    
    bool isHappy(int n) {
        set<int> res;
        while(res.find(n)==res.end()){
            if(n==1) return true;
            res.insert(n);
            n=total_sum(n);          
        }
        return false;
    }
};
```
