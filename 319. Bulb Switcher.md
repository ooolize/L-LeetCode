### 题目
[Bulb Switcher](https://leetcode-cn.com/problems/bulb-switcher/submissions/)
### 思路
模拟n次开关过程，数据量很大，超时27/35
### 代码
模拟超时
```c++
class Solution {
public:
    int bulbSwitch(int n) {
        if(n==1||n==0) return n;
        const int k=n;
        bitset<100000> b;
        for(int i=0;i<n;i++)
            b.set(i);
        
        for(int i=1;i<n;i++){
            for(int j=0;j<n;j++){
                if((j+1)%(i+1)==0) b.flip(j);   
            }
        }
        return b.count();
    }
};
```
