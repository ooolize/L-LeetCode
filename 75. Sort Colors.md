### 题目
[75. Sort Colors](https://leetcode-cn.com/problems/sort-colors/submissions/)
### 思路
思路一是基数排序.
+ 先一次遍历nums用num储存数字出现的次数。 

+ 再遍历num就地改变nums

思路二
### 代码
```c++
lass Solution {
public:
    void sortColors(vector<int>& nums) {
        vector<int> num(3,0);
        for(auto &p:nums)  num[p]++;
        int i=0;
        for(int k=0;k<num.size();k++){
            while(num[k]--) nums[i++]=k;
        }
        return;
    }
};
```
