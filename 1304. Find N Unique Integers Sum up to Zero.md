### 题目
[1304. Find N Unique Integers Sum up to Zero](https://leetcode-cn.com/problems/find-n-unique-integers-sum-up-to-zero/submissions/)
### 思路
8ms 91.69%

### 代码
```c++
class Solution {
public:
    vector<int> sumZero(int n) {
        vector<int> v;
        int k=n/2;
        for(int i=1;i<=k;i++) {
            v.push_back(i);
            v.push_back(-i);
        }
        if(n%2) v.push_back(0);
        return v;
    }
};
```
