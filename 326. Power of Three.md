### 题目
[Power of Three](https://leetcode-cn.com/problems/power-of-three/submissions/)
### 思路
16ms 90.02%
就是循环直到不小于n,每次采用位运算实现乘三操作。
### 代码
```c++
class Solution {
public:
    bool isPowerOfThree(int n) {
        long tmp=1;
        while(tmp<n){
            tmp+=(tmp<<1);
        }
        return tmp==n;
    }
};
```
