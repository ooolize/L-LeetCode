### 题目
[441. Arranging Coins](https://leetcode-cn.com/problems/arranging-coins/submissions/)
### 思路
20ms 36.76%

n层有n个，计算前n层共有多少个。找到第一个大于n的层数cur，然后返回cur-1.
### 代码
```c++
int arrangeCoins(int n) {
        int cur=0;
        long total=0;
        while(total<(long)n){
            cur++;
            total=(long)cur*((long)cur+1)/2;
        }
        if(total==n)return cur;
        else return cur-1;
    }
```
