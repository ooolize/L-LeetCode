### 题目
[367. Valid Perfect Square](https://leetcode-cn.com/problems/valid-perfect-square/submissions/)
### 思路
4ms 71.49%

逐个求平方数直到它大于num。
+ 注意乘法溢出，需要强制转化
### 代码
```c++
class Solution {
public:
    bool isPerfectSquare(int num) {
        int i=1;
        long n=0;
        
        while(n<num){
            n=long(i)*long(i);
            i++;
        }
        return num==n;
    }
};
```
